What is TOPSIS?

TOPSIS (Technique for Order Preference by Similarity to Ideal Solution) is a multi-criteria decision-making method that ranks alternatives based on their geometric distance from the ideal best and ideal worst solutions.

How TOPSIS Works

  1. Normalize the decision matrix using vector normalization
  2. Calculate weighted normalized matrix using criterion weights
  3. Determine ideal best and ideal worst solutions
  4. Compute separation distances from ideal solutions
  5. Calculate TOPSIS scores (0 to 1, higher is better)
  6. Rank alternatives in descending order of scores

Key Concepts

Benefit Criteria: Higher values are better (e.g., Quality, Speed)

Cost Criteria: Lower values are better (e.g., Price, Risk)

Weights: Relative importance of each criterion (will be normalized to sum to 1)

Score: Ranges from 0 to 1, where 1 = closest to ideal solution
